*
Данные в исходниках soc_znach<год>_.<doc/docx/pdf> по субъектам РФ содержат информацию за текущий и предыдущий года. 
При этом, если сравнивать в текущем файле значение предыдщего года со значением из предыдущего файла, то данные могут различаться.
В выгрузке "по субъектам РФ.csv" для текущего года содержится информация из текущего файла, без уточнения значения из следующего файла.

Пример: 
Брянская область,число пациентов с впервые в жизни установ.диагнозом,на 100000 соот.населения,2016,"53,4",soc_znach2016.docx - значение из текущего файла
Брянская область,число пациентов с впервые в жизни установ.диагнозом,на 100000 соот.населения,2016,"53,6",soc_znach2017.docx - значение из файла следующего года
