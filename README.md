# IrkutskProject
Medical Statistics Visualization

Данные для построения дашбордов предоставлены заказчиком в виде с сводных таблиц excel формы 32, где 2016 год имеет старый формат данных, года 2017-2024 имеют новый общий формат данных. 

1) Задача первого этапа сведена к тому, чтобы преобразовать данные из сводных таблиц формата excel в длинный формат csv, удобный для анализа. Формат и названия заголовков предоставлены заказчиком.
   
Для конвертации в csv формат разработаны:
- 2100.ipynb - конвертирует данные таблицы 2100 в длинный формат csv
- 2200.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2300.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2400.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2500.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2600.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2700.ipynb - конвертирует данные таблицы 2200 в длинный формат csv
- 2800.ipynb - конвертирует данные таблицы 2200 в длинный формат csv

Для корректной работы необходимо:
- создать папку с годом
- выложить файл сводного отчета в папку Original в соответствующем году,
- прописать названия файла, год в скрипты (в будущем данный шаг можно автоматизировать)
- выполнить скрипты и получить сформированные csv файлы в папке Common.
